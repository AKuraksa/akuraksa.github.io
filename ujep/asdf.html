Nastavení OSPF na Turris Omnia

Jednoduchý průvodce pro správu OSPF pomocí Quagga

1. Přístup k routeru

Připojte se k routeru Turris Omnia přes SSH nebo webové rozhraní.

2. Instalace balíčků

Zajistěte, aby byl nainstalován balíček pro OSPF:

opkg update
opkg install quagga

3. Konfigurace Quagga

Po instalaci Quagga upravte konfiguraci:

Soubor zebra.conf:

hostname Router
password vaše_heslo
enable password vaše_enable_heslo

Soubor ospfd.conf:

hostname Router
password vaše_heslo
log file /var/log/quagga/ospfd.log

router ospf
ospf router-id 1.1.1.1
network 192.168.1.0/24 area 0.0.0.0

4. Spuštění Quagga

/etc/init.d/quagga enable
/etc/init.d/quagga start

5. Ověření stavu OSPF

Zkontrolujte stav OSPF a sousední routery:

vtysh
show ip ospf neighbor

6. Firewall

Ujistěte se, že firewall povoluje provoz OSPF (UDP port 89).
